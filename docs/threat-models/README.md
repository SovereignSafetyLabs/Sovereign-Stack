# Threat Model Documents

## Scope

This directory contains **analytical threat model documents**
related to the Vesta Governance Framework.

These materials explore hypothetical failure modes, adversarial dynamics,
and governance stress scenarios for research and evaluation purposes.

## Posture

All documents in this directory are **research and reference materials only**.
They are non-operational and are not intended for deployment, certification,
or real-world threat mitigation use.

## Exclusions

This directory does not contain:
- Implementation guides
- Deployment instructions
- Operational threat assessments
- Performance guarantees
- Safety assurances

## Authority

See repository root `README.md`, `ARTIFACTS.md`, and `VALIDATION_STATUS.md`
for authoritative information about scope, classification, and status.
