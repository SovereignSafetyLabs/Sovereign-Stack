# Core Framework Specifications

## Scope

This directory contains **architectural and design-level specification documents**
for Vesta Governance Framework components.

These materials describe conceptual structures, interfaces, and governance logic
for research and analytical purposes.

## Posture

All documents in this directory are **research and reference materials only**.
They are non-operational and are not validated for deployment or production use.

## Exclusions

This directory does not contain:
- Implementation guides
- Deployment instructions
- Operational claims
- Performance guarantees
- Safety assurances

## Authority

See repository root `README.md`, `ARTIFACTS.md`, and `VALIDATION_STATUS.md`
for authoritative information about scope, classification, and status.
